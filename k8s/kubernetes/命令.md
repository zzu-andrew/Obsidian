
## `kubectl`







### `kubectl get`

- 获取一个节点所有的 `yaml` 配置

```bash
$ $kubectl get pod prometheus-k8s-0 -n base-services -o yaml
```





### `kubectl describe` 输出节点的详细信息

- 输出命名空间位`base-services`节点类型为`statefulset`类型节点的详细信息
```bash
$ kubectl describe statefulset prometheus-k8s -n base-services
```
